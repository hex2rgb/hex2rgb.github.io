开机挂载smb、cifs

2019-03-06 · 21 个字 · 1 分钟 读完
linux
smb · cifs · 挂载硬盘
# 手动挂载
mount.cifs //192.168.123.234/download/outer-source /mnt/data -o username=username,password=password

# 开机挂载
# 编辑 /etc/fstab
# vers=1.0,_netdev 服务端版本低客户端增加该配置，以及网络等待配置
//192.168.123.234/download/outer-source /mnt/data cifs vers=1.0,username=username,password=password,_netdev,uid=0,gid=0,iocharset=utf8 0 0
